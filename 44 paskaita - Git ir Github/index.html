Git ir Github

Git'as - kodo versijavimo programa.
GitHub'as - versijuoto kodo talpykla internete.

Git komandos

  • init - sukurti git repozitoriją dabartiniame aplankale
  • status - pateikia git repozitorijos dabartinę informaciją
  • add - prideda nurodytus failus prie commit'inimui paruoštų failų (stage'ina)
  • rm --cacheed || restore --staged - išima nurodytus failus ir stage'inimo zonos(unstage'ina)
  • commit -m "žinutė apie commit'ą" - sukuria kodo versiją iš stage'intų failų ir prideda žinutę apie pakitimus (žinutė turėtų būti trumpa ir aiški)
